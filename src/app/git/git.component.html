Comandos de Git
  1. git init: Inicializa un nuevo repositorio Git en el directorio actual.
  2. git clone: Clona un repositorio Git existente en tu máquina local.
  3. git add: Agrega cambios al área de preparación (staging) para ser incluidos en el próximo commit.
  4. git commit: Crea un nuevo commit con los cambios en el área de preparación.
  5. git status: Muestra el estado actual de tu repositorio, incluyendo cambios pendientes y archivos sin seguimiento.
  6. git branch: Muestra la lista de ramas (branch) y resalta la rama actual.
  7. git checkout: Cambia entre ramas o commits previos.
  8. git pull: Obtiene cambios del repositorio remoto y los fusiona con tu rama local.
  9. git push: Envía tus cambios locales al repositorio remoto.
  10. git merge: Combina una rama con otra.

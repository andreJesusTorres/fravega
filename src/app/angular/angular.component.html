Instalación de Angular

Para comenzar con Angular, primero debes instalar Angular CLI (Command Line Interface) globalmente en tu sistema. Asegúrate de tener Node.js instalado. Abre una terminal y ejecuta el siguiente comando:

npm install -g @angular/cli

Luego, puedes crear un nuevo proyecto Angular usando el siguiente comando:

ng new nombre-del-proyecto
Creación de Componentes en Angular

En Angular, los componentes son la base de la aplicación. Puedes generar un nuevo componente con el siguiente comando:

ng generate component nombre-del-componente

Esto generará archivos relacionados con el componente, incluyendo HTML, CSS, TypeScript, y automáticamente actualizará tu archivo app.module.ts para registrar el componente.

Comandos Básicos en Angular
  • ng serve: Iniciar el servidor de desarrollo de Angular.
  • ng build: Compilar la aplicación Angular para producción.
  • ng generate: Generar componentes, módulos, servicios, etc.
  • ng test: Ejecutar pruebas unitarias.
  • ng lint: Analizar el código en busca de problemas de estilo y calidad.
¿Qué es Angular?

Angular es un framework de desarrollo de aplicaciones web y móviles desarrollado por Google. Permite la creación de aplicaciones web dinámicas y de una sola página (SPA) de manera eficiente. Angular utiliza TypeScript como lenguaje de programación y se basa en el concepto de componentes para organizar y estructurar la aplicación.

Uso del enrutamiento en Angular (Opcional)

El enrutamiento en Angular permite la navegación entre diferentes vistas o componentes de la aplicación. Para habilitar el enrutamiento, puedes configurar rutas en el archivo app-routing.module.ts y utilizar el módulo RouterModule para crear enlaces y facilitar la navegación entre componentes. Esto es útil para crear aplicaciones multipágina o para dividir la aplicación en diferentes secciones.
